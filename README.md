# Guess The Word Game

## Summary

This game gets 10 words from the dictionary using the library RandomWords which is then taken and entered into the library PyMultiDictionary. This then finds the definitions in the EDUCALINGO dictionary.You are given the 10 words in a random order one at a time, along with the definitions and have to guess each of them. The asterisks hide the word to guess used in the dictionary definition. the goal is to guess all 10 words before running out of lives. You will be given a score out of 10 at the end.
